init

git init

Creates new .git repository folder in the current folder.

status

git status

Returns status of current branch including:

  • Untracked files: files which are have not being added to the repository.
  • Unstaged files: files that are tracked but which are have not been staged.
  • Staged files: files that are staged and will be on next commit.

add

git add <file1 name> [file2 name] ...

  1. Adds specified file(s) to repository branch if file is not currently being tracked.

  2. Stages modified files that are alreay part of current repository branch.

Separate files with spaces.

Sepecifying a directory will add all files to next commit.

git add -A

Adds all changes including new files, modified files and removed files in current directory.

git add .

Adds all changes including new files, modified files and but not removed files in current directory.
